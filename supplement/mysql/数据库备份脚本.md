# 数据库脚本备份

- windows下借助bat脚本

  ```cmd
  @echo off
  set "Ymd=%date:~,4%%date:~5,2%%date:~8,2%"
  cd  "C:\Program Files"\MySQL\"MySQL Server 5.0"\bin
  net stop MySQL
  xcopy C:\ProgramData\MySQL\"MySQL Server 8.0"\Data\zixishi\*.*   D:\MySQLdata_Bak\mysql\%Ymd%\ /s /y
  net start MySQL
  ```

  - 如果路径中有空格，则要用引号包含
  - 写好脚本之后，设置系统的定时任务
